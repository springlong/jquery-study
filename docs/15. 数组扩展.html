1. $.makeArray(nodes)

    将参数nodes（类型为NodeList）的值转换为Array类型。（返回类型：Array）

    NodeList会随着DOM的更改而被更新，在某些情况下不宜直接操作，建议转换为静态数组后再行处理。

    $.type(document.getElementsByTagName("p"))

    $.type($.makeArray(document.getElementsByTagName("p")))

  2. Array.unique()

    返回目标数组去除重复值之后所组成的新数组（原数组的值不受影响）。（返回类型：Array）

    [1, 2, 3, 4, 5].unique()

    [1, 2, 3, 3, 4, 5, 4, 3, 2, 1].unique()

  3. Array.indexOf(val)

    返回目标数组中参数val的值第一次出现所在的索引位置，不存在则返回-1。（返回类型：Number）

    参数val：需要进行查询的值；

    [1, 2, 3, 4, 5].indexOf("1")

    [1, 2, 3, 4, 5].indexOf("a")

    [1, 2, 3, 4, 5].indexOf(1)

    [1, 2, 3, 4, 5].indexOf(5)

  4. Array.filter(function(item, i){})

    遍历目标数组中的每一个元素，筛选出符合条件（即回调函数返回ture时）的所有成员并以新数组的形式返回。（返回类型：Array）

    回调函数-参数item：当前项的值；

    回调函数-参数i：当前项的索引值；

    回调函数-this：window对象；

    [1, 2, 3, 4, 5].filter(function(item, i){ return i < 2; })

    [1, 2, 3, 4, 5].filter(function(item, i){ return item > 2; })

    [1, 2, 3, 4, 5].filter(function(item, i){ return item > 5; })

    [1, 2, 3, 4, 5].filter(function(item, i){ return item % 2 === 0; })

    [1, 2, 3, 4, 5].filter(function(item, i){ return item % 2 !== 0; })

  5. Array.forEach(function(item, i){})

    遍历目标数组中的每一个元素，并执行回调函数。（返回类型：undefined）

    回调函数-参数item：当前项的值；

    回调函数-参数i：当前项的索引值；

    回调函数-this：window对象；

    注：下面的示例代码将遍历目标数组，并在每一行的结果元素（ID命名为“forEach1”-“forEach5”）中输出每一项的值。

    [6, 7, 8, 9, 10].forEach(function(item, i){ $("#forEach" + ++i).html(item); })

     

     

     

     

  6. Array.remove(val)

    删除原数组中与参数val的值相等的所有元素，并返回原数组（原数组中的值将会受到影响）。（返回类型：Array）

    参数val：需要删除的值；

    [1, 2, 3, 4, 5, 5].remove("3")

    [1, 2, 3, 4, 5, 5].remove(3)

    [1, 2, 3, 4, 5, 5].remove(5)
