1. $.each(obj, function(i, item){})

    遍历数组或对象，并将每一项的索引和值参与回调函数的执行。（返回类型：undefined）

    参数obj：需要进行遍历的数组或对象；

    回调函数-参数i：当前项的索引值；

    回调函数-参数item：当前项的值；

    回调函数-this：当前项的值；

    var obj = ["a", "b", "c"];
$.each(obj, function(i, item)
{
    alert("当前项：" + i + "；值：" + item + "；this：" + this);
});


    var obj = {name: "Jerry", sex: "male", age: 22};
$.each(obj, function(i, item)
{
    alert("当前项：" + i + "；值：" + item + "；this：" + this);
});
